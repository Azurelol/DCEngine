class  MetaData
 A MetaData object is a non-templated class, allowing them to be stored in a data structure. There is one MetaData object for each type of data registered to the system, and the MetaData object of a corresponding type is a representation of that type. It stores information about whether or not it is a plain old data type (POD), the size of the type, its members, methods, and name. Inheritance information can be stored along with multiple inheritance information (though the latter is not implemented). More...
