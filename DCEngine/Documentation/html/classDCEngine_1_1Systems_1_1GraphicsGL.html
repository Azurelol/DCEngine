void DCEngine::Systems::GraphicsGL::ConfigureSpriteTextVAO ( )

Configures the VAO used for SpriteText.

Note
We reserve enough memory when initiating the VBO so that we can later update the VBO's memory when rendering characters. THe 2D quad requries 6 vertices of 4 floats each so we reserve 6 * 4 floats of memory. Because we will be updating the content of the VBO's memory quite often, we allocate with GL_DYNAMIC_DRAW.
