file  Font.cpp
 The Font resource handles the loading and processing of a TrueType font from file. It uses the 'Freetype' library to do so. Freetype loads these fonts and for each glyph generates a bitmap image and caculates several metrics. We can extract these bitmap images for generating textures and position each character glyph appropiately using the loading metrics.
